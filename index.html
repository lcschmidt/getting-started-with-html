The Basics of the Web and HTML.

The major pieces of the web are:
1. your browser - which displays files found on the web,
2. the Internet - the world's largest computer network,
3. HTTP - the main protocol of the web, and
4. servers - computers that host the files that make up the web.

HTML (hypertext markup language) is made up of:
+ text content (what you see)
+ mark up (what it looks like)
+ references to other documents (eg images & videos)
+ links to other pages

HTML mark up is a way to change how the content of the website looks. An HTML element consists of opening tags and closing tags around contents (like text). One example of mark up is making text bold, like the title above. To do this, you type < b > text < /b > .

HTML elements can have attributes which provide additional information about the element. For example, the anchor tag is used for making links. If I want to include a link to the Udacity website by allowing you to click on the word Udacity in this sentence, I write < a href = "www.udacity.com" > Udacity < /a >.
